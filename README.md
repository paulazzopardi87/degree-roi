# UK Careers Intelligence

An interactive guide to 200 UK careers, analysed across four dimensions:

- **AI Resilience** — how resistant each career is to automation (0–100)
- **Median Salary** — entry, mid and senior ranges
- **Degree ROI** — payback period using the Gemini formula (tuition + opportunity cost vs salary premium)
- **No-Degree Entry** — realistic probability of entering without a degree

## Features

- Search and filter by sector, degree type, AI resilience, ROI and no-degree entry
- Card and table views
- Click any career for a full breakdown including the ROI calculation
- Personalise with your actual loan debt to recalculate real payback years

## Data Sources

Salary data synthesised from ONS ASHE, Glassdoor UK, LinkedIn Salary and professional body surveys (2024/25). Regional variation applies — London salaries typically 20–40% above medians shown.
